About This Project

This project contains examples of commonly used HTML tags, attributes, and elements. Each file focuses on a specific HTML concept.

HTML Topics

forms.html

Complete reference for HTML forms including input types, form controls, validation attributes, and form elements.

tables.html

Learn HTML tables, rows, columns, headers, captions, rowspan, colspan, and table-related elements.

lists.html

Examples of ordered lists, unordered lists, description lists, and nested lists.

media.html

Working with images, audio, video, SVG, canvas, and embedded content.

semantic.html

HTML5 semantic elements like header, nav, main, section, article, aside, and footer.

links.html

Anchor tag usage including external links, internal links, email links, and downloads.

images.html

Image elements, attributes, responsive images, image maps, and related tags.

audio-video.html

HTML audio and video elements with their attributes and source handling.

iframe.html

Embedding external pages, videos, maps, and other documents using iframe.
